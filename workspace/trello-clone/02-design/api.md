# API — trello-clone

There is no network API (per `02-design/architecture.md`, this is a client-only, `localStorage`-backed app). The "API" is the set of public, in-process JS functions exported by `store.js` (state + mutations) and `storage.js` (persistence), called by `events.js`/`app.js` and consumed by `render.js`. Every function below is synchronous and returns/mutates the in-memory state object `{boards, lists, cards}` (each entity: `{id, ...fields, order}`; `id`s are strings generated by the store, e.g. via `crypto.randomUUID()`).

## store.js — state & mutations

### `getState()`
- **Serves:** FR1-FR14 (read path for all rendering)
- **Input:** none
- **Output:** current state object `{boards: Board[], lists: List[], cards: Card[]}`
- **Errors:** none
- **Example:** `render.js` calls `store.getState()` on every re-render to paint the boards/lists/cards view.

### `createBoard(title)`
- **Serves:** FR1
- **Input:** `title: string` (non-empty, trimmed)
- **Output:** the new `Board {id, title, order}`
- **Errors:** throws `Error('title required')` if `title` is empty after trim
- **Example:** `store.createBoard('Home Renovation')` → adds a board and persists it.

### `renameBoard(boardId, title)`
- **Serves:** FR1
- **Input:** `boardId: string`, `title: string` (non-empty, trimmed)
- **Output:** updated `Board`
- **Errors:** throws `Error('board not found')` if `boardId` unknown; `Error('title required')` if empty
- **Example:** `store.renameBoard('b1', 'Renovation 2026')`.

### `deleteBoard(boardId)`
- **Serves:** FR1, FR11 (confirmation happens in `events.js` before calling this)
- **Input:** `boardId: string`
- **Output:** `void` (also cascades: deletes all `lists`/`cards` belonging to the board)
- **Errors:** throws `Error('board not found')` if `boardId` unknown
- **Example:** `store.deleteBoard('b1')` after user confirms deletion.

### `reorderBoards(boardId, toIndex)`
- **Serves:** FR9
- **Input:** `boardId: string`, `toIndex: number` (0-based target position among boards)
- **Output:** `void` (mutates `order` field of affected boards)
- **Errors:** throws `Error('board not found')`; clamps `toIndex` to `[0, boards.length - 1]`
- **Example:** `store.reorderBoards('b1', 0)` moves board `b1` to the first position.

### `createList(boardId, title)`
- **Serves:** FR2
- **Input:** `boardId: string`, `title: string` (non-empty)
- **Output:** the new `List {id, boardId, title, order}`
- **Errors:** throws `Error('board not found')`; `Error('title required')`
- **Example:** `store.createList('b1', 'To Do')`.

### `renameList(listId, title)`
- **Serves:** FR2
- **Input:** `listId: string`, `title: string` (non-empty)
- **Output:** updated `List`
- **Errors:** throws `Error('list not found')`; `Error('title required')`
- **Example:** `store.renameList('l1', 'Doing')`.

### `deleteList(listId)`
- **Serves:** FR2, FR11
- **Input:** `listId: string`
- **Output:** `void` (cascades: deletes all `cards` in the list)
- **Errors:** throws `Error('list not found')`
- **Example:** `store.deleteList('l1')` after confirmation.

### `reorderLists(boardId, listId, toIndex)`
- **Serves:** FR6
- **Input:** `boardId: string`, `listId: string`, `toIndex: number`
- **Output:** `void` (mutates `order` of lists within that board)
- **Errors:** throws `Error('list not found')`; clamps `toIndex` to valid range
- **Example:** `store.reorderLists('b1', 'l2', 0)` moves list `l2` to the front of board `b1`.

### `createCard(listId, title)`
- **Serves:** FR3
- **Input:** `listId: string`, `title: string` (non-empty)
- **Output:** the new `Card {id, listId, title, description: '', order, label?, dueDate?}`
- **Errors:** throws `Error('list not found')`; `Error('title required')`
- **Example:** `store.createCard('l1', 'Buy paint')`.

### `renameCard(cardId, title)`
- **Serves:** FR3
- **Input:** `cardId: string`, `title: string` (non-empty)
- **Output:** updated `Card`
- **Errors:** throws `Error('card not found')`; `Error('title required')`
- **Example:** `store.renameCard('c1', 'Buy blue paint')`.

### `updateCardDescription(cardId, description)`
- **Serves:** FR8
- **Input:** `cardId: string`, `description: string` (may be empty)
- **Output:** updated `Card`
- **Errors:** throws `Error('card not found')`
- **Example:** `store.updateCardDescription('c1', 'Two coats, satin finish')` from the card-detail modal save action.

### `deleteCard(cardId)`
- **Serves:** FR3, FR11
- **Input:** `cardId: string`
- **Output:** `void`
- **Errors:** throws `Error('card not found')`
- **Example:** `store.deleteCard('c1')` after confirmation.

### `reorderCard(cardId, toListId, toIndex)`
- **Serves:** FR4 (same-list reorder when `toListId` equals current list), FR5 (cross-list move), FR6a (click-based "Move to..." fallback calls this same function)
- **Input:** `cardId: string`, `toListId: string`, `toIndex: number`
- **Output:** `void` (mutates `listId` and `order` of affected cards)
- **Errors:** throws `Error('card not found')`; `Error('list not found')`; clamps `toIndex` to valid range
- **Example:** `store.reorderCard('c1', 'l2', 2)` — drag handler drops card `c1` into list `l2` at position 2.

### `setCardLabel(cardId, label)`
- **Serves:** FR12
- **Input:** `cardId: string`, `label: string | null` (a color/tag name; `null` clears it)
- **Output:** updated `Card`
- **Errors:** throws `Error('card not found')`
- **Example:** `store.setCardLabel('c1', 'red')`.

### `setCardDueDate(cardId, dueDate)`
- **Serves:** FR13
- **Input:** `cardId: string`, `dueDate: string | null` (ISO date `YYYY-MM-DD`; `null` clears it)
- **Output:** updated `Card`
- **Errors:** throws `Error('card not found')`; `Error('invalid date')` if `dueDate` set and not a valid ISO date
- **Example:** `store.setCardDueDate('c1', '2026-08-01')`.

### `filterCards(boardId, keyword)`
- **Serves:** FR14
- **Input:** `boardId: string`, `keyword: string` (case-insensitive substring match against card `title`/`description`)
- **Output:** `Card[]` matching cards (does not mutate state)
- **Errors:** throws `Error('board not found')`
- **Example:** `store.filterCards('b1', 'paint')` — `events.js` calls this on search-input change and `render.js` shows only matches.

## storage.js — persistence

### `load()`
- **Serves:** FR7 (restore on boot)
- **Input:** none
- **Output:** the persisted state object, or a fresh `{boards: [], lists: [], cards: []}` if no data / wrong schema version is found in `localStorage`
- **Errors:** none (corrupt/unparseable JSON is caught internally and treated as "no data")
- **Example:** `app.js` calls `storage.load()` once at boot and passes the result to `store` as initial state.

### `save(state)`
- **Serves:** FR7 (write-through on every mutation)
- **Input:** `state: {boards, lists, cards}`
- **Output:** `void`
- **Errors:** none thrown to caller; a `localStorage` quota/availability failure is caught and logged, app continues to work in-memory for the session
- **Example:** `store.js` calls `storage.save(state)` internally after every mutating function above.

## app.js — boot

### `init()`
- **Serves:** FR1-FR14 (bootstraps everything), NFR3 (works immediately, no login)
- **Input:** none
- **Output:** `void` — loads state via `storage.load()`, calls `render.js`'s initial render, wires up `events.js` listeners
- **Errors:** none (propagates only unexpected runtime errors, none anticipated)
- **Example:** `index.html`'s `<script type="module" src="js/app.js">` calls `init()` on module load.
